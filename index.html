Our Method: Graph Definition & Traversal

üìä Graph Definition

We model the blockchain snapshot as an undirected cyclic graph:

  • Vertices: Wallets or smart contracts
  • Edges: Transactions between them
  • Why Undirected? The flow of value can be ambiguous. A wallet can initiate a transfer that causes a contract to send tokens to that same wallet, making the direction of value transfer complex.

üîç Graph Traversal

We use the Gremlin query language to traverse the graph, starting from the ERC-20 contract under analysis to find all cyclic paths of a specific length.

Gremlin Query & Technology Choices

üîç The Gremlin Query

g.V()
  .has('account','hex', '0x98b4ca8bd52e4ed1f28d3f30d9f567d1166c9483')
  .as('a')                                                  // select iFish contract
  .bothE().bothV()                                         // traverse edges
  .repeat(simplePath()).times(2)                          // repeat 2 iterations
  .where(out().as('a'))                                       // filter cycles back to iFish
  .bothE().dedup().subgraph('subg').cap('subg').next()    // export subgraph

ü§î Why Not Modern Python Libraries?

  • Performance at Scale: Gremlin/TinkerPop offers superior performance for large-scale graph traversals with built-in parallelization and MapReduce support
  • Database Integration: Native graph database integration allows efficient storage and querying of blockchain data at enterprise scale
  • Gephi Visualization: Gephi's Force Atlas 2 algorithm provides specialized graph layout optimized for financial network analysis, with better complexity O(N log N)
  • Domain Expertise: Graph databases are specifically designed for this type of traversal problem, while Python libraries are more general-purpose

The "Funding-Origin Indirection Metric" (Œπ)

üéØ Purpose

To quantitatively confirm the visual findings from the graph.

üí° Concept

It measures the proportion of wallets funding an ICO that share a common ancestor wallet at various levels of indirection.

üîç Why This Matters

In legitimate ICOs, funding typically comes from diverse, unrelated sources. In pump-and-dump schemes, multiple wallets often trace back to the same origin, creating artificial trading volume.

Case Study: The iFishYunYu Attack

üìÖ Background

In July 2018, Ethereum gas prices spiked 500% due to over 80,000 transactions involving the iFishYunYu ERC-20 token.

ü§î The Suspicions
